# 🎯 Project Overview

In today’s competitive landscape, retaining customers is more cost-effective than acquiring new ones.  
This project provides a comprehensive **End-to-End Data Analytics solution** to understand customer attrition for a Telecom firm.  

By transforming raw data into actionable insights, we identify:
- Why customers leave  
- Who is at risk  

---

# 🚀 Key Goals

- 🔄 **Full ETL Pipeline**  
  Cleaned and processed raw customer data into a structured database.

- 📊 **Interactive Dashboard**  
  Visualized demographics, geography, and service patterns.

- 🎯 **Churn Profiling**  
  Identified high-risk segments to guide marketing campaigns.

- 🔮 **Predictive Insight**  
  Established a foundation for predicting future churners.

---

# 🛠️ Tech Stack

- **Data Processing:** SQL / Excel (ETL Process)  
- **Visualization:** Power BI  
- **Analysis:**  
  - Churn Rate  
  - Revenue Impact  
  - Customer Lifetime Value (CLV)

# 📈 Key Metrics & Features

The analysis focuses on four critical pillars:

| Pillar        | Focus Areas |
|--------------|------------|
| 👥 Demographics | Age groups, gender distribution, and marital status |
| 🌍 Geographic   | Churn analysis by state and region |
| 💳 Account Info | Payment methods, contract types, and tenure |
| 📡 Services     | Fiber optics vs. DSL, streaming habits, and support plans |

---

## 💡 Core Metrics Calculated

- **Total Customers** → Overview of the entire user base  
- **Churn Rate** → Percentage of customers leaving the service  
- **New Joiners** → Tracking growth alongside attrition  

---

# 🧠 Conclusion

This project demonstrates how data-driven decisions can reduce churn. By analyzing the Churner Profile, the business can now implement targeted loyalty programs and improve service offerings where they matter most.
